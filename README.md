我内心渴望可以向那些萌系画师一样画一幅精良的，震撼的，或是美丽的人设，赛博朋克街景，又或者是同人图，而非能够开发卷积神经网络或者进行几个统计学测试；

我畅想着某天放声歌唱，不用为了高音而降低一个声部，去唱那些我未曾能够嘹亮完成的乐章，而非使用geopandas的种种功能实现geo data可视化

或许有一天，我可以在贝加尔湖畔坐船在风景优胜处停靠，用厚重的单反记录宇宙的不确定，地球的天造，三维的壮丽，而非利用线性代数在vector space求解难题。

我对我仍然保持吉他的一些技巧而感到高兴，因为随着长大我越发热爱音乐。

我内心更偏向文科，这不是指历史，政治和地理，而是更加传统的，更贴近生活的唱歌，画画，写作，以及摄影。

如果有可能，我希望让上帝给予我一份和现在不大相同的礼物，或许包裹里装着一盒水粉颜料，一支钢笔，一个麦克风，一把乐器以及一台单反。但我仍感谢上帝曾经给予我的礼物是计算机，那个曾经的人类科技结晶。
